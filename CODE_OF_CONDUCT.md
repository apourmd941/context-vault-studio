# Code of Conduct

## Our Standard

We want this project to feel serious, curious, kind, and safe for contributors across experience levels.

Examples of behavior that contributes to a positive environment:

- assuming good intent
- giving actionable technical feedback without hostility
- documenting tradeoffs clearly
- welcoming questions from new contributors

Examples of unacceptable behavior:

- harassment or intimidation
- insulting or demeaning comments
- bad-faith sabotage
- publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or other contributions that violate this code of conduct.

## Reporting

Report conduct issues privately using the process in `SECURITY.md` until a dedicated community contact is published.
